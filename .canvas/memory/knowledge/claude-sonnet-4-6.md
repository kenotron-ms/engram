# Claude Sonnet 4.6 Release Information

## What We Learned

Claude Sonnet 4.6 was released on February 17, 2026. It represents a major upgrade of the Sonnet model line with improvements across:
- Coding capabilities (70% preference over Sonnet 4.5, 59% over Opus 4.5)
- Computer use (near human-level on complex tasks)
- Long-context reasoning (1M token context window in beta)
- Agent planning and knowledge work
- Design sensibility

## Key Specifications

- **Pricing:** $3/$15 per million tokens (same as Sonnet 4.5)
- **Context Window:** 1M tokens (beta)
- **Availability:** Default model for Free and Pro plans on claude.ai

## Benchmark Performance

- SWE-bench Verified: 80.2%
- ARC-AGI-2: 60.4%
- Terminal-Bench 2.0: Leading performance
- Matches Opus 4.6 on OfficeQA for document comprehension

## Why It Matters

This model brings near-Opus-level performance at Sonnet pricing, making advanced capabilities more accessible. Particularly strong for coding tasks that previously required more expensive Opus models.
