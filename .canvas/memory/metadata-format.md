# Memory System Metadata Format

## YAML Frontmatter
The Canvas memory system uses YAML frontmatter for metadata in all memory files.

### Standard Fields
- `type`: Category of knowledge (technical, preference, decision, etc.)
- `keywords`: Array of searchable terms
- `created`: ISO date of creation
- Additional fields as needed per memory type
